Unfortunately, quality doesn't dictate popularity, but popularity would dictate the number of libraries, right?

  • mgl-mat
  • magicl

and perhaps a few others, their functionality is still dwarfed by the more popular languages. So, without using projects like cffi, cl-autowrap, py4cl, py4cl2-cffi, it can be hard if not impossible to meet one's needs while staying in Common Lisp.
